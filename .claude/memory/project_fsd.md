This is a learning project where the user is studying Feature Sliced Design (FSD) and applying it to a comfy-store tutorial that was not originally built with FSD.

**Why:** User is actively learning FSD and using this project as a hands-on exercise.

**How to apply:** Always suggest and structure code according to FSD principles — layers (`app`, `pages`, `widgets`, `features`, `entities`, `shared`), slices, and segments. When the user asks where to put something or how to structure something, answer in FSD terms. Flag any existing code that deviates from FSD and suggest how to align it.
